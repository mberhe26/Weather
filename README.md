# Web Development Project 5 - *Weather*

Submitted by: **Mastewal Berhe**

This web app: **simple weather application that allows users to search for and view current weather information for a list of cities. The app fetches data from the OpenWeatherMap API and displays the city name, temperature, wind speed, and weather description. It also provides summary statistics like the total number of cities, highest wind speed, and lowest wind speed.**

Time spent: **24** hours spent in total
